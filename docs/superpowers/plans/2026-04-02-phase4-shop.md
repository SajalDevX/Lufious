# Phase 4: Shop (Marketplace)

**Goal:** Build the full marketplace — browse listings, create a listing, view listing detail, and wishlist toggle — all backed by Firestore.

**Architecture:**
- Listings live at `marketplace/listings/{listingId}` (global collection)
- Wishlists live at `users/{uid}/wishlist/{listingId}` (per-user subcollection, doc ID = listingId)
- No photo upload in this phase (photoUrls stored as empty list; can add Firebase Storage later)
- `CreateListing` and `ListingDetail` are sub-screens within the inner tab NavHost (bottom nav visible, consistent with Garden sub-screens)
- Category filter: All · Plants · Pots & Planters · Soil & Fertilizers · Tools · Accessories

**Tech Stack:** Firestore, Dagger Hilt, Jetpack Compose, Navigation
